# Como usar os posts do Instagram

Este guia explica como usar o conteúdo criado para o Instagram da WM Som e Luz.

---

## Estrutura dos posts

Cada post está organizado em uma pasta com os seguintes arquivos:

- **legenda-curta.txt**: Legenda resumida (use quando quiser postar rápido)
- **legenda-longa.txt**: Legenda completa com detalhes e CTA
- **hashtags.txt**: Hashtags relevantes para o post
- **prompt-imagem.txt**: Prompt para gerar imagens com IA (Midjourney, DALL-E) ou orientação para Canva/Figma
- **roteiro-telas.txt** (para carrosséis): Descrição de cada slide do carrossel
- **roteiro-cenas.txt** (para reels): Descrição de cada cena do vídeo

---

## Passo a passo para postar

### 1. Escolha o post do dia
Veja o arquivo `calendario.md` para saber qual post publicar hoje.

### 2. Crie as imagens/vídeos
- **Para carrosséis e fotos:** Use Canva, Figma ou Adobe Express. Siga o roteiro em `roteiro-telas.txt` ou use o prompt em `prompt-imagem.txt` para gerar imagens base com IA.
- **Para reels:** Filme ou edite vídeos seguindo `roteiro-cenas.txt`. Use CapCut, InShot ou Adobe Premiere.

### 3. Escreva a legenda
- Copie o conteúdo de `legenda-longa.txt` (recomendado) ou `legenda-curta.txt`.
- Personalize se necessário (exemplo: adicionar data específica de evento).

### 4. Adicione as hashtags
- Copie o conteúdo de `hashtags.txt` e cole no final da legenda ou no primeiro comentário.

### 5. Publique
- Publique no Instagram no horário ideal (11h-13h ou 18h-20h).

## Guia de estilo visual

### Paleta de cores (do logo WM)
- **Verde:** Verde-amarelado (W do logo)
- **Azul:** Azul escuro (M do logo)
- **Roxo:** Tons de transição (ondas)
- **Acentos:** Rosa neon, amarelo dourado (vibe anos 80-90)
- **Fundo:** Escuro ou neutro para destacar elementos

### Elementos visuais
- Bola de discoteca
- Ondas sonoras pulsantes
- Silhouetas de pessoas dançando
- Raios de luz (feixe de iluminação)
- Discos de vinil (anos 70-80)
- Telão com videoclipes
- Equipamentos (caixas de som, moving lights, painel de LED)

### Estética geral
- **Vibe:** Retrô-futurista, nostálgico (anos 70-90), mas profissional
- **Tom:** Festivo, animado, energia de pista de dança
- **Fontes:** Bold, sans-serif, fácil de ler
- **Composição:** Limpa, moderna, com hierarquia visual clara

## Temas sugeridos para novos posts
- Depoimento de cliente (foto + depoimento em texto)
- Antes/depois (local vazio → festa montada)
- Equipamento em destaque (moving lights, telão LED, etc.)
- Diferenciais (1 por post): videoclipes, VDJ, personalização, qualidade técnica
- 4 frentes (1 post por frente): Casamento, Formaturas, Aniversários, Corporativo
- FAQ (1 pergunta por post)
- Bastidores (equipe montando, operando, desmontando)
